# 🎨 The Canvas Theory – Art Landing Page

A visually engaging and responsive **Art Landing Page** built using **HTML, CSS, and JavaScript**. This project focuses on combining creativity with clean UI design, smooth animations, and interactive elements.

## 📌 Features

* 🎨 Modern and aesthetic UI design
* 🖼️ Responsive image gallery layout
* ⚡ Smooth scroll animations using AOS
* ⌨️ Custom **typewriter effect** for showcasing art types
* 🎯 Interactive hover effects & transitions
* 🎛️ Gradient text effects for visual appeal
* 🧩 Reusable components and clean structure
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* **HTML5** – Page structure
* **CSS3** – Styling, Flexbox, animations, gradients
* **JavaScript** – Typewriter effect & interactivity
* **AOS Library** – Scroll animations
* **Font Awesome** – Icons

---

## 📚 What I Learned

* Structuring a complete landing page using semantic HTML
* Building responsive layouts using **Flexbox**
* Creating advanced UI with **gradients, transitions, and hover effects**
* Implementing **scroll-based animations (AOS)**
* Writing a **typewriter animation from scratch using JavaScript**
* Improving UI/UX with spacing, typography, and visual hierarchy
* Enhancing design with micro-interactions and animations
* Making the layout responsive using **media queries**
